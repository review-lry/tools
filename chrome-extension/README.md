# 🛠️ 开发者工具箱 - Chrome 插件

一个强大的 Chrome 浏览器插件，提供常用开发工具，支持快捷键操作和页面数据处理。

## ✨ 功能特性

### 🚀 快捷操作（一键完成）
- **格式化页面JSON** - 自动格式化当前页面的 JSON 数据
- **复制选中内容** - 快速复制页面选中的文本
- **当前时间戳** - 一键复制当前时间戳
- **生成UUID** - 一键生成并复制 UUID

### 🕐 时间工具
- 当前时间实时显示
- 时间戳 ↔ 日期 转换
- 日期转时间戳（支持多种格式）

### 🔐 编解码
- Base64 编码/解码
- URL 编码/解码
- HTML 转义/反转义

### 📝 格式化
- JSON 格式化/压缩
- JWT Token 解析

### 📄 文本处理
- 文本统计（字符数、中文字数、单词数、行数）
- 大小写转换
- 排序/去重

### 🎲 生成器
- UUID 生成
- 随机密码生成
- 二维码生成

### 🔒 哈希
- SHA-256 / SHA-512 哈希生成

## 📦 安装方法

### 方法一：开发者模式安装（推荐）

1. **下载插件**
   ```bash
   git clone https://github.com/review-lry/tools.git
   # 或直接下载 chrome-extension 文件夹
   ```

2. **打开 Chrome 扩展管理**
   - 在 Chrome 地址栏输入：`chrome://extensions/`
   - 或：菜单 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 右上角开关打开「开发者模式」

4. **加载插件**
   - 点击「加载已解压的扩展程序」
   - 选择 `chrome-extension` 文件夹
   - 完成！

### 方法二：拖拽安装（需解压）

1. 将 `chrome-extension` 文件夹解压到本地
2. 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 直接将文件夹拖入浏览器窗口

## 🎯 使用场景

### 场景1：格式化 API 返回的 JSON
1. 打开 API 响应页面
2. 点击插件图标或按 `Ctrl+Shift+D`
3. 点击「格式化页面JSON」
4. 页面自动变成格式化的 JSON

### 场景2：转换时间戳
1. 复制一个时间戳
2. 按 `Ctrl+Shift+D` 打开插件
3. 粘贴到时间戳输入框
4. 点击「转换」

### 场景3：快速复制当前时间戳
1. 按 `Ctrl+Shift+D`
2. 点击「当前时间戳」
3. 自动复制到剪贴板

### 场景4：生成随机密码
1. 打开插件 → 生成 → 随机密码
2. 设置长度和字符类型
3. 点击「生成」

## ⚙️ 权限说明

插件需要以下权限：

| 权限 | 用途 |
|------|------|
| `activeTab` | 获取当前标签页信息 |
| `clipboardWrite` | 写入剪贴板 |
| `clipboardRead` | 读取剪贴板 |
| `storage` | 存储设置 |
| `scripting` | 在页面中执行脚本 |

## 🔧 自定义图标（可选）

如果需要自定义图标：

1. 准备 3 个尺寸的 PNG 图标：16x16、48x48、128x128
2. 放入 `icons/` 文件夹
3. 更新 `manifest.json` 中的图标路径

或使用在线工具生成：
- https://favicon.io/
- https://realfavicongenerator.net/
