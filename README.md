# 🛠️ 开发者工具箱

一个集成了多种常用开发工具的在线工具箱，帮助开发者提高工作效率。

## ✨ 工具列表

### ⏰ 时间工具
- **当前时间** - 实时显示当前时间和时间戳
- **时间戳转换** - 时间戳转日期时间
- **日期转时间戳** - 日期时间转时间戳

### 📝 文本处理
- **JSON 格式化** - JSON 格式化和压缩
- **Base64 编解码** - Base64 编码和解码
- **URL 编解码** - URL 编码和解码
- **文本统计** - 统计字符数、单词数、行数等

### 🔐 加密与哈希
- **MD5 生成** - 生成文本的 MD5 哈希值
- **JWT 解析** - 解析 JWT Token 的 Header 和 Payload

### 🎨 视觉工具
- **颜色转换** - Hex、RGB、HSL 颜色格式转换

### 🔢 数据处理
- **进制转换** - 二进制、八进制、十进制、十六进制互转
- **UUID 生成器** - 批量生成 UUID

### 🔍 其他工具
- **正则表达式测试** - 测试正则表达式匹配

## 🚀 使用方法

1. 打开 `index.html` 文件即可在浏览器中使用
2. 所有工具均在本地运行，无需网络连接（除了 CDN 加载 CryptoJS 库）

## 🎯 特性

- 📱 响应式设计，支持桌面和移动端
- 🎨 美观的渐变 UI 设计
- ⚡ 实时处理，无需刷新页面
- 📋 一键复制结果
- 🔒 完全本地运行，保护隐私

## 📦 技术栈

- HTML5
- CSS3 (渐变、Grid 布局)
- 原生 JavaScript
- CryptoJS (MD5 加密)
