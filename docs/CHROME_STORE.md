## 📤 发布步骤

1. 登录 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. 支付 $5 一次性注册费
3. 点击 "Add new item"
4. 上传 .zip 文件
5. 填写商店详情
6. 提交审核

### 审核时间
- 通常: 24-48 小时
- 最长: 7 天

---

## 🔄 后续更新

发布后更新流程:
1. 修改代码
2. 更新 manifest.json 版本号
3. 重新打包 zip
4. 在 Developer Dashboard 上传新版本
5. 提交审核

更新审核通常比首次发布快。
